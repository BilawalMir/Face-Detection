# Real-Time Face Detection

This project uses OpenCV to perform real-time face detection using your webcam.

## Requirements

- Python 3.10
- OpenCV (`cv2`)

## How to Run

1. Install OpenCV if you haven't:
   ```bash
   pip install opencv-python
   ````

2. Run the script:

   ```bash
   python main.py
   ```

3. Press **a** to exit the video window.
